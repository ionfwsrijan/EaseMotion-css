# Multi Select Chips - EaseMotion Submission

A pill-shaped multi-select chip component built with pure HTML, CSS, and vanilla JS. Click chips to toggle a selected state with animated background fill and checkmark reveal.

---

## Features

- Pill-shaped toggle chips with smooth background and border transitions
- Checkmark icon scales in with a spring-like cubic-bezier curve on selection
- Hover lift effect for interactive feedback
- Fully responsive flex-wrap layout
- No external dependencies (Google Fonts only for Inter)

---

## Files Included

- `demo.html` → UI structure and toggle logic
- `style.css` → styling, transitions, and key animations
- `README.md` → documentation

---

## Preview Behavior

- Click any chip to toggle `.msc-selected`
- Selected chips fill with blue, border darkens, text turns white
- A checkmark circle appears with a scale bounce animation
- Unselected chips return to their default outline style
