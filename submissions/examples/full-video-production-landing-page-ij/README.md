# Full Video Production Company Landing Page

1. What does this do? A cinematic video production landing page with showreel-style hero, services grid, portfolio gallery, process timeline, client logos, testimonials, and quote CTA built entirely with EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — the page uses `ease-card-hover`, `ease-reveal-up`, `ease-fade-in`, `ease-slide-up`, `ease-reveal-scale`, `ease-hover-glow`, `ease-grid-cols-*`, and dozens of other EaseMotion classes for all layout, components, and animations.

3. Why is it useful? Demonstrates EaseMotion's ability to create a cinematic, portfolio-driven creative agency experience with zero custom CSS animations — all scroll reveals, hover effects, and layout grids use composable framework classes.
