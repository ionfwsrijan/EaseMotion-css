# Alphabet Sort Animate

1. **What does this do?** Animates a shuffled set of alphabet letters into sorted order with scale, translate, and swap keyframe effects on each letter tile.
2. **How is it used?** Place letters inside `class="alphabet-sort__rack"` as `class="alphabet-sort__letter"` elements; trigger sort, shuffle, or reset via JavaScript by toggling animation classes like `--entering` and `--swapping`.
3. **Why is it useful?** It demonstrates a playful, variable-driven sorting visualization that fits EaseMotion CSS's philosophy of using CSS custom properties and keyframe animations for interactive, performant UI motion.
