# Ease Orbit Loader

## Overview

Ease Orbit Loader is a futuristic loading animation inspired by atomic structures, where multiple glowing particles orbit around a central core at different speeds.

---

## Features

- 🪐 Multiple orbiting particles
- ⚛️ Atomic-inspired animation
- ✨ Glowing particle effects
- 🔄 Different orbit speeds
- 💯 Pure HTML & CSS
- 📱 Responsive demo

---

## Demo

The demo showcases:

- Central glowing core
- Three animated orbital rings
- Three glowing particles
- Infinite loading animation

---

## Usage

```html
<div class="loader">
    ...
</div>
```

---

## Customization

You can customize:

- Orbit sizes
- Rotation speed
- Particle colors
- Glow intensity
- Core size

---

## Browser Support

Works in all modern browsers supporting CSS transforms and animations.
