# Analog Clock (`ease-analog-clock-rotate-ij`)

An analog clock with continuously rotating hands built with **HTML5**, **Pure CSS**, and minimal JavaScript.

## Features

- Smooth CSS-driven hand rotation
- Hour, minute, and second hands
- Hour tick marks with major indicators
- CSS custom properties for theming
- Minimal JS for initial time alignment
- No external dependencies

## CSS Custom Properties

| Variable         | Default    | Description             |
|------------------|------------|-------------------------|
| `--clock-bg`     | `#1a1a2e`  | Clock face background   |
| `--hand-color`   | `#4ecdc4`  | Primary hand color      |
| `--tick-duration`| `1s`       | Base timing multiplier  |
