# Alert Zone Highlight

1. **What does this do?** Creates a pulsing alert zone with a glowing border, icon pulse, and radial glow that draws attention to critical notifications.
2. **How is it used?** Apply `class="alert-zone alert-zone--active"` to a container; toggle the `--active` class to start or stop the alert animation.
3. **Why is it useful?** It provides an attention-grabbing yet unobtrusive alert pattern that aligns with EaseMotion's accessible, variable-driven animation approach.
