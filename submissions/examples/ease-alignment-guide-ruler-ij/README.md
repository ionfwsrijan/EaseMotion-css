# Alignment Guide Ruler

1. **What does this do?** Displays crosshair alignment rulers with dashed tick marks, a center dot, and a pulse target ring, all fading in sequentially.
2. **How is it used?** Wrap a canvas area with `class="alignment-guide"` and place ruler, crosshair, and target elements inside; toggle `--guide-duration` to control the reveal speed.
3. **Why is it useful?** It offers a polished design-tool-style alignment overlay that reinforces EaseMotion's commitment to utility-driven, variable-customizable CSS animations.
