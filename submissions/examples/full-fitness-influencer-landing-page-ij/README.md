# Fitness Influencer/Coach Landing Page

1. What does this do? A full landing page for a fitness influencer or coach featuring a hero with transformation stats and program CTA, a programs grid (beginner/intermediate/advanced), client before/after transformation gallery, about the coach section, testimonials with measurable results, social proof stats, and a free consultation CTA. All motion uses EaseMotion CSS classes only.

2. How is it used? Open demo.html in a browser. The page uses EaseMotion CSS framework classes for all animations: ease-fade-in, ease-slide-up for entrance sequences, ease-reveal with reveal.js for scroll-triggered reveals, ease-hover-lift-shadow for program card hover effects, and ease-bounce-in for stat counters. The showReveal.js helper triggers stats on scroll.

3. Why is it useful? Fitness coaches need landing pages that build trust through visual proof (transformations) and social proof (stats, testimonials). The smooth scroll-triggered reveals keep potential clients engaged as they browse, while the energetic entrance animations reinforce the transformation narrative. The staggered card animations make program comparisons intuitive.
