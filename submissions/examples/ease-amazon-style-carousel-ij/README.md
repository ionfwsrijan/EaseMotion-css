# Amazon-Style Product Carousel (`ease-amazon-style-carousel-ij`)

A responsive product carousel built with **HTML5**, **Pure CSS**, and minimal JavaScript.

## Features

- Smooth slide transitions
- Previous / Next navigation arrows
- Dot indicators with active state
- Responsive (3 columns desktop, 2 columns mobile)
- CSS custom properties for easy theming
- No external dependencies

## CSS Custom Properties

| Variable          | Default    | Description           |
|-------------------|------------|-----------------------|
| `--card-bg`       | `#2d2d44`  | Card background color |
| `--arrow-color`   | `#4ecdc4`  | Arrow and accent color|
| `--slide-duration`| `0.4s`     | Slide transition speed|
