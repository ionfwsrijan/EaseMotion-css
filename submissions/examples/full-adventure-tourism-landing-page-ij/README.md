# Adventure Tourism Landing Page

1. What does this do? A full landing page for an adventure tourism company featuring a high-energy hero section, tour packages grid (hiking, rafting, climbing) with difficulty indicators, guide profiles, safety certification badges, customer adventure story testimonials, and a booking CTA. All animations use EaseMotion CSS framework classes.

2. How is it used? Open demo.html in a browser. The page uses EaseMotion CSS classes (ease-fade-in, ease-slide-up, ease-reveal, etc.) for scroll-triggered and entrance animations. The reveal.js script handles IntersectionObserver-based scroll reveals. Section cards use ease-hover-lift-shadow for hover effects and ease-hover-glow for badge interactions.

3. Why is it useful? Adventure tourism sites need high-energy visuals to convey excitement and trust. The staggered entrance animations create a dynamic browsing experience that keeps potential customers engaged, while the reveal-based scroll animations guide the user naturally through the content hierarchy from hero to CTA.
