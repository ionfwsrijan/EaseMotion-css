# Marketing Agency Landing Page

1. What does this do? A full landing page for a results-driven marketing agency called RocketLab Agency. Contains a bold hero with stats grid (client revenue, avg ROAS, campaigns launched, rating), four services (SEO, PPC, Social, Content), three case study cards with real metrics, a client logos grid, a four-phase process timeline, testimonial cards with ROI callouts, and a free audit CTA. All animations use EaseMotion CSS classes (ease-slide-up, ease-reveal-up, ease-bounce-in, ease-card-hover, etc.).

2. How is it used? Open demo.html in any browser. The page references EaseMotion's easemotion.css for all animation and component classes. The reveal.js script activates scroll-triggered entrances. Customize the stats, service descriptions, case studies, and logos in demo.html and style.css to match a specific agency.

3. Why is it useful? Marketing agencies need a high-impact landing page that demonstrates authority and results. This template uses EaseMotion animations to create an energetic, data-forward feel — stat cards pulse, services bounce in, case studies reveal on scroll — all without custom keyframes. Responsive and ready to deploy.
