Animated Scroll to Top

A smooth scroll-to-top button with bounce entrance and hover rotation. Scroll down to see it in action.

Why use a scroll-to-top button?

Long-form content benefits from quick navigation aids. A scroll-to-top button saves users from manual scrolling, improving UX on article-heavy pages.

How it works

The button listens for the scroll event. Once the page scrolls past a threshold (e.g., 400px), it gains the .ast-visible class which triggers a fade-in bounce via CSS. On click, window.scrollTo({ top: 0, behavior: 'smooth' }) animates back up.

Customisation

All visual properties — colours, sizes, timing — are controlled through CSS custom properties in :root, making it easy to adapt to your design system.

Performance notes

Only opacity and transform are animated, both GPU-accelerated properties. The scroll handler is kept lightweight to avoid jank.

Accessibility

The button carries aria-label="Scroll to top" and is keyboard-focusable. The smooth-scrolling behaviour respects the user's prefers-reduced-motion setting.

Browser support

Works in all modern browsers. scroll-behavior: smooth is supported in Chrome, Firefox, Edge, Safari 15.4+. For older browsers, a window.scrollTo(0, 0) fallback applies.
