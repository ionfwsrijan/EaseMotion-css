# Ease Radar Sweep

## Overview

Ease Radar Sweep is a futuristic radar animation utility built using pure HTML and CSS. A rotating scan line sweeps around the radar while glowing targets pulse to simulate object detection.

---

## Features

- Rotating radar sweep animation
- Three concentric radar rings
- Animated glowing targets
- Infinite smooth animation
- Pure HTML & CSS
- Responsive centered layout

---

## Demo

- Radar sweep
- Multiple target indicators
- Pulsing signal animation

---

## Usage

```html
<div class="radar">

<div class="line"></div>

<div class="dot"></div>

</div>
```

---

## Customization

You can customize:

- Radar size
- Sweep speed
- Glow color
- Number of target dots
- Ring spacing

---

## Browser Support

Works in all modern browsers supporting CSS animations and transforms.
