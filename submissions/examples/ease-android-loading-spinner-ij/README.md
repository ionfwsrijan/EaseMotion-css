# Android-Style Loading Spinner (`ease-android-loading-spinner-ij`)

An Android-style circular loading spinner and indeterminate progress bar built with **HTML5** and **Pure CSS**.

## Features

- Indeterminate circular spinner (default, small, large)
- Custom color support via CSS variables
- Adjustable spin speed
- Indeterminate loading bar
- Pulsing loading text
- No JavaScript
- No external dependencies

## CSS Custom Properties

| Variable           | Default    | Description              |
|--------------------|------------|--------------------------|
| `--spinner-color`  | `#4ecdc4`  | Spinner active color     |
| `--track-color`    | `#2d2d44`  | Spinner track color      |
| `--spin-duration`  | `1s`       | Full rotation duration   |
